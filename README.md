- **Modern Design**: Clean, minimal UI with dark theme and blue accents

## 🔧 Local Development

To run this project locally:

1. Clone the repository:
```bash
git clone https://github.com/raju2026/portfolio.git
cd portfolio
```

2. Open `index.html` in your browser, or use a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js (if available)
npx serve .
```

3. Visit `http://localhost:8000` in your browser.

## 🚀 Deployment

### Option 1: GitHub Pages (Current)
The portfolio is currently deployed on GitHub Pages at:
`https://raju2026.github.io/portfolio`

### Option 2: Render Deployment

To deploy on Render:

1. **Connect Repository**:
   - Go to [Render.com](https://render.com)
   - Sign up/Login with your account
   - Click "New" → "Static Site"
   - Connect your GitHub account
   - Select the `raju2026/portfolio` repository

2. **Configure Build Settings**:
   - **Build Command**: `echo "No build required"`
   - **Publish Directory**: `./` (root directory)
   - **Node Version**: Latest LTS (not required for static sites)

3. **Deploy**:
   - Click "Create Static Site"
   - Render will automatically deploy your portfolio
   - Get your live URL (e.g., `https://portfolio.onrender.com`)
